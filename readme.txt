# RAG Machine Learning Project

A Retrieval-Augmented Generation (RAG) application built using Python and LLM technologies.

## Features
- Document ingestion
- Vector search
- LLM-powered question answering
- Retrieval pipeline

## Tech Stack
- Python
- LangChain
- FAISS
- OpenAI/Groq/Ollama
- Streamlit

## Installation

pip install -r requirements.txt

## Run

python app.py
